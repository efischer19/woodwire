Welcome

A minimal starter built with vanilla HTML, CSS, and JavaScript. Edit src/index.html, src/assets/styles.css, and src/scripts/app.js to build your application.

No build step required — open this file directly in a browser.

Features

  • Semantic HTML

    Proper landmark elements and heading hierarchy for accessibility.

  • Responsive CSS

    Mobile-first layout with custom properties and no framework dependencies.

  • Dark Mode

    Accessible theme toggle with OS preference detection and persistence.

  • Keyboard Navigable

    Skip-to-content link and visible focus indicators throughout.
